# AGENTS.md

Packaged desktop app icons for Tauri across macOS, Windows, and Linux.

## Folder Role
- Contains `.png`, `.ico`, and `.icns` assets referenced by Tauri config/build tooling.
- Files are packaging assets, not runtime UI components.

## Editing Notes
- Replace icons as a coordinated set so all platforms remain consistent.
- Keep expected filenames unless also updating Tauri config.
- Do not hand-edit binary icon files; regenerate from source artwork when needed.
